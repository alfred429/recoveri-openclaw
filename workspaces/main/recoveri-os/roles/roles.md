# ROKOS Roles

ROKOS (Recoveri Operating Kernel OS) defines roles for agents.

## Active Agents

- **Jarvis (CoS)**: Board gateway, routing.
- **Optimus (CEO)**: Decisions, compliance.
- **Data (CTO)**: Tech, code.
- **Alpha (CRO)**: Revenue, content.
- **Kitt (COO)**: Ops, finance, security.
- **Oracle (Athena) (Consultant)**: Deep research. Expensive.

## Others
- **Mozart (Chef)**: Orchestration.

Athena = Oracle (alias for deep research agent).
