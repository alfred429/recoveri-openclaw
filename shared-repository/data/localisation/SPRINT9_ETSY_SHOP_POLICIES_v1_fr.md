## Politique de Retour

Etsy ne permet pas aux vendeurs de définir des politiques de retour formelles pour les annonces numériques. Les remboursements sont à la discrétion du vendeur. C'est ce que les clients voient :

### Texte de Politique Recommandé pour la Boutique :

**Téléchargements Numériques – Positionnement sur les Retours**

Étant donné la nature d'accès instantané des produits numériques, nous n’offrons pas de remboursements automatiques pour les téléchargements. Cependant, votre satisfaction est importante pour nous.

Si vous rencontrez l'une des situations suivantes, veuillez nous contacter et nous vous aiderons :
- Problèmes techniques empêchant l'accès à vos fichiers
- Fichiers qui ne correspondent pas à la description de l'annonce
- Téléchargements incomplets ou corrompus
- Achats multiples accidentels

Nous répondons à tous les messages dans un délai de 2 heures et travaillerons avec vous pour trouver une solution. La plupart des problèmes sont résolus en une seule interaction.

Si nous ne pouvons pas résoudre votre problème de manière technique, nous examinerons les demandes de remboursement au cas par cas.

### Logique Interne (non destinée au client) :
- Les remboursements sont décidés uniquement par le Chef (Cadre CS v2, Compétence 132)
- Les agents CS utilisent le Modèle C1 (déboguer en premier, ne jamais promettre)
- Escalade au Chef via Telegram pour toutes les demandes de remboursement
- La Protection d’Achat Etsy couvre jusqu’à 250 € si l’article ne correspond pas à la description
- Période de litige : le client peut ouvrir un dossier s'il n’a pas de réponse dans un délai de 48h

## Méthode de Livraison

### Texte de Politique Recommandé pour la Boutique :

**Téléchargement Numérique Instantané**

Tous les produits sont livrés sous forme de téléchargements numériques instantanés via Etsy. Après l'achat :

1. Vous recevrez un e-mail de confirmation contenant un lien de téléchargement.
2. Vos fichiers sont également disponibles dans votre compte Etsy sous la section « Achats ».
3. Les liens de téléchargement sont valides pendant 30 jours à compter de l'achat.
4. La plupart des produits sont livrés sous forme de fichier ZIP contenant tous les formats inclus.

Aucun article physique ne sera expédié. La livraison est immédiate au traitement du paiement (les paiements PayPal peuvent prendre quelques minutes supplémentaires).

### Informations sur les Formats de Fichiers (inclure pour chaque annonce) :
- Indiquez clairement tous les formats de fichiers inclus (PDF, XLSX, lien vers Google Docs, etc.)
- Indiquez les logiciels compatibles (par exemple, « Compatible avec Microsoft Excel 2016+, Google Sheets, Numbers»)
- Indiquez les appareils compatibles (par exemple, « Windows, Mac, iOS, Android»)

## Conditions Générales des Produits Numériques

### Texte Recommandé pour la Boutique :

**Ce que Vous Achetez**

Lorsque vous achetez un produit numérique de RecoveriStudio, vous recevez une licence d'utilisation personnelle pour télécharger et utiliser les fichiers inclus dans l'annonce.

Ce qui est inclus :
- Tous les fichiers énumérés dans la description du produit
- Mises à jour gratuites pour cette version du produit (si applicable)
- Accès à notre équipe de support client

Ce qui n'est pas inclus :
- Droits de revente ou de redistribution
- Licence commerciale (contactez-nous pour les demandes commerciales)
- Modifications ou services de conception personnalisés (nous pouvons vous orienter vers des ressources)

Tous les produits sont testés sur plusieurs appareils et systèmes d'exploitation avant d'être mis en vente. Si vous avez des questions de compatibilité, veuillez nous contacter avant d'acheter – nous serons ravis de vous aider.

## Confidentialité / Conformité RGPD

### Texte Recommandé pour la Boutique :

RecoveriStudio est basé au Royaume-Uni. Nous respectons le RGPD britannique et le RGPD européen pour tous les données des clients.

Nous ne collectons que :
- Les informations fournies via la plateforme Etsy pour la gestion des commandes
- Les adresses e-mail volontairement fournies via notre inscription au site web

Nous ne vendons, ne partageons ni ne distribuons les données des clients à des tiers. Pour notre politique de confidentialité complète, veuillez consulter recoveri.io/privacy (quand sera disponible).
