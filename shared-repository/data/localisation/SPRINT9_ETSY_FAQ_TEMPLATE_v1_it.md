## FAQ Standard (copiare in ogni annuncio, adattare i campi tra parentesi per il prodotto)

### Come posso scaricare i miei file dopo l'acquisto?

Dopo il pagamento, Etsy vi invierà una email con il link di download. Potete anche trovare i vostri file andando nel vostro account Etsy > Acquisti > trovare il vostro ordine > cliccare su “Scarica File”. I link di download sono validi per 30 giorni. Se avete pagato con PayPal, potrebbe volerci un po' più di tempo per l'elaborazione. Controllate la cartella spam se non ricevete l'email.

### In quali formati file sono disponibili i file?

Questo prodotto include: [ELENCA FORMATI – es. PDF, XLSX, link a Google Docs]. Avrete bisogno di [ELENCA SOFTWARE – es. Microsoft Excel 2016 o versioni successive, Google Sheets, Apple Numbers] per aprire e utilizzare i file. Compatibile con [ELENCA DISPOSITIVI – es. Windows, Mac, iOS, Android].

### La cartella ZIP contiene i file – come posso aprirla?

Su Windows: cliccare con il tasto destro sul file ZIP e selezionare “Estrai Tutto”. Su Mac: fare doppio clic sul file ZIP e i file verranno estratti automaticamente. Se riscontrate problemi, riprovate a scaricare il file o utilizzate uno strumento gratuito come 7-Zip. Avete bisogno di aiuto? Scriveteci e vi guideremo passo dopo passo.

### Posso modificare questo prodotto?

[SE EDITABILE]: Sì! Questo prodotto è completamente modificabile in [SOFTWARE]. Potete modificare il testo, i colori, la disposizione e aggiungere il vostro branding. Apri il file in [SOFTWARE] e iniziate a personalizzarlo.

[SE NON EDITABILE]: Questo è un file [FORMATO] pronto all'uso. È progettato per essere utilizzato così com'è per [CASO d'USO]. Se avete bisogno di una versione modificabile, controllate il nostro negozio per alternative personalizzabili o contattateci per ricevere consigli.

### Cosa succede se ho bisogno di aiuto o ho un problema?

Scriveteci tramite Etsy. Rispondiamo a tutti i messaggi entro 2 ore e la maggior parte dei problemi viene risolta in un'unica interazione. Che si tratti di un problema di download, di una domanda di compatibilità o di qualsiasi altra cosa, siamo qui per aiutarvi.

### Offrite rimborsi per prodotti digitali?

A causa della natura immediata dell'accesso ai download digitali, rivediamo attentamente le richieste di rimborso. Se riscontrate un problema tecnico o il prodotto non corrisponde alla descrizione dell'annuncio, contattateci e lavoreremo con voi per trovare una soluzione. La vostra soddisfazione è importante per noi.
