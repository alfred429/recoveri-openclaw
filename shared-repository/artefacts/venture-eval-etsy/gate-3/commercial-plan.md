### 3. Pricing Strategy
**Builds on Gate 2 (£5-7 single / £12-15 bundles / £15-20 premium).** 65% margins post-Etsy fees (6.5% + £0.16 listing + 3% proc.).

| Tier | Price Range | Products | Margin (post-fees, £7 ex.) | Upsell Path |
|------|-------------|----------|-----------------------------|-------------|
| **Single** | £5-7 | Basic templates/trackers (e.g., #3, #9) | £4.00-£5.00 (70-80%) | → Bundle (+£7 value) |
| **Bundle** | £12-15 | 3-5 items (e.g., #2, #6) | £9-£11 (70%) | → Premium (+£5 value) |
| **Premium** | £15-20 | Adaptive/full systems (e.g., #4, #10) | £11-£15 (65-75%) | Custom add-on (£10+) |

**Dynamic Pricing:** Test A/B via Etsy (e.g., £6 vs £7 singles). Volume discounts (buy 3 = 10% off). Seasonality: +20% Q4 peaks.

**Avg. Price Target:** £7 (Gate 2 projection). 13,650 Year 1 units → £95k base revenue.
